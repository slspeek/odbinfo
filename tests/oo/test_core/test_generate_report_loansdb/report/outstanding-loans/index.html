Outstanding Loans in diagram

Command 
SELECT "Loans"."Item" as "Item",
       "Loans"."LentToPerson" as "LentToPerson",
       "Loans"."Returned" as "Returned",
       "Loans"."DateExpected" as "DateExpected",
       "People"."Landline"
FROM   "Loans", "People"
WHERE  "Loans"."LentToPerson" = "People"."Name"
   and "Loans"."Returned" = 0
   and "Loans"."DateExpected" < current_date
ORDER BY "Loans"."LentToPerson"
